## AI-Powered Marathi News Admin Panel

This project is a secure, AI-powered admin panel built using the MERN stack to streamline the creation, optimization, and publishing of Marathi news articles.

Editors can input raw news data (Marathi or English), generate well-structured Marathi news articles using Google Gemini AI, analyze and improve SEO, manage article images with captions and alt text, and push finalized content directly to WordPress as drafts.

### Key Features
- Secure authentication with JWT and role-based access (Admin / Editor)
- AI-based Marathi news generation using Google Gemini
- SEO analysis and AI-powered SEO improvement (meta title, description, keywords)
- Image upload with preview, featured image selection, captions & alt text
- WordPress REST API integration for draft publishing
- Clean and lightweight React admin UI
- Built for newsroom-style editorial workflows

### Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- AI: Google Gemini API
- CMS Integration: WordPress REST API

This project is designed for private newsroom use and focuses on editorial accuracy, SEO best practices, and secure content workflows.
