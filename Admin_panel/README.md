## AI-Powered Marathi News Admin Panel

This project is a secure, AI-powered admin panel for streamlining the creation, optimization, and publishing of Marathi news articles.

Editors can input raw news data (Marathi or English), generate well-structured Marathi news articles using local manus_agents AI, analyze and improve SEO, manage article images with captions and alt text, and push finalized content directly to WordPress as drafts.

### Key Features
- Secure authentication with JWT and role-based access (Admin / Editor)
- AI-based Marathi news generation using local manus_agents
- SEO analysis and AI-powered SEO improvement (meta title, description, keywords)
- Image upload with preview, featured image selection, captions & alt text
- WordPress REST API integration for draft publishing
- Clean and lightweight React admin UI
- Built for newsroom-style editorial workflows
- Multi-agent system with Ollama for local, cost-free AI processing

### Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- Database: MySQL with Sequelize ORM
- AI: Local manus_agents (Ollama-based, zero API costs)
- CMS Integration: WordPress REST API

This project is designed for private newsroom use and focuses on editorial accuracy, SEO best practices, secure content workflows, and cost-effective AI processing.
